# Ggulook Game Studio

다양한 미니 게임을 즐길 수 있는 게임 허브 프로젝트입니다.

## 프로젝트 개요
- **플랫폼 이름**: GGULOOK STUDIO
- **핵심 구조**: 메인 메뉴(게임 선택기) + 개별 게임 화면 분리
- **디자인 컨셉**: 모던한 카드형 UI, 다크/화이트 모드 완벽 지원

## 현재 구현된 게임
### 1. Ggulook Jump (기존 부엉이 점프)
- **장르**: 무한 러너 / 액션
- **특징**: 이단 점프 가능, 점진적 난이도 상승, 히트박스 보정 적용
