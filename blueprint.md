# 부엉이 점프 게임 (Owl Jump Game)

오른쪽에서 다가오는 장애물을 점프해서 피하는 액션 게임입니다.

## 프로젝트 개요
- **캐릭터**: 기존의 부엉이 SVG 아이콘 활용
- **핵심 메카닉**: 중력 적용 점프, 무한 스크롤 장애물, 충돌 감지
- **디자인 컨셉**: 모던한 다크/화이트 모드 지원, 부드러운 애니메이션, 직관적인 UI
